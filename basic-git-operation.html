Basic Git Operations




Status


Step 1: Go to Terminal by clicking the Toogle Panel in your repository in VS Code.


Step 2: Type in git status.



Add


Step 1: Go to Terminal in VS Code.


Step 2: Type in git add and what you want to add.



Commit


Step 1: Go to Terminal in VS Code


Step 2: Type in git commit -m and whatever message you want to write after in quotation marks.



Push


Step 1: Go to Terminal in VS Code


Step 2: Type in git push and what you want to push



Pull


Step 1: Go to Terminal in VS Code.


Step 2: Type in git pull and what you want to pull.
