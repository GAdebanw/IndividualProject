GitHub Repository



Creating a New Repository on GitHub.



Step 1: Login into you GitHub account.


Step 2: Click on the + located in the right corner scroll to New Repository.


Step 3: Name your New Repository.





Cloning the Repository Page Using VSCode Intergrated Git Support



Step 1: Open VSCode.


Step 2: Click on the Source Control located in the navgation bar on the left side.


Step 3: Cick on the Clone Repository button.


Step 4: Copy and paste the git url you got when you create a New Repository.


Step 5: Choose which folder you would like to clone your repository.


Step 6: It will ask to you to login in to your GitHub and connect VSCode and GitHub together accept it.


Step: It will now open up in VSCode.
