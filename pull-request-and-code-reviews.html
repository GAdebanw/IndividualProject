Pull Request and Code Reviews with VS Code



Create Pull Request Using VS Code



Step 1: Go to the Source Control.


Step 2: Type in what you want to pull.


Step 3: Click on the pull request button located beside the checkmark in source control panel.


Step 4: It will ask to create a name for the pull request.


Step 5: Type in the name for your pull request.


Step 6: It will promote to create a branch.


Step 7: Type in the name of the branch and press enter.


Step 8: It will display a pop up that said a pull request has been create and you can switch to branch.


Step 9: Click Switch to Branch.


Step 10: It will Switch to that branch.


Step 11: Then you can go and make any changes for that pull request.




Creating Pull Request Using GitHub



Step 1: Click the pull request button locted in the navbar.


Step 2: Click the create tab.


Step 3: Type pull request in the search bar and press Enter.


Step 4: Click where you want the pull request.




Review and Dicuss Pull Requests



Step 1: Click pull request in your repository page.


Step 2: Find and Click the pull request you like to review.


Step 3: Click the file change located in the pull request.


Step 4: Review the pull request.


Step 5: Find the line you would like to make a comment.


Step 6: Then hover over it and click on the blue comment and type your comment.


Step 7: Click start a review.
