Branching with VS Code



How to Create Branches in VS Code



Step 1: Go to the bottom of in VSCode


Step 2: Click in the thing that looks like a branch and has the word main beside it.


Step 3: Click + Create new branch.


Step 4: Name the branch.


OR


Step 1: Go to Terminal in VS Code.


Step 2: Type in git branch and the name of the branch




How to switching branches with VS Code



Step 1: Go to Terminal in VS Code.


Step 2: Type in git switch and the name of the branch you want to switch to.


OR


Step 1: Go to Terminal in VS Code.


Step 2: Type in git checkout and the name of the branch you want to switch to.




Branching to Fix Bugs



Step 1: Go to Terminal in VS Code


Step 2: Type in git checkout -b emergency-fix
